# AnestLog — App de Controle de Procedimentos para Residentes de Anestesiologia

## Sobre o projeto
App web (React + Next.js + Tailwind CSS) para residentes de anestesiologia registrarem cirurgias, procedimentos e acompanharem sua evolução com gráficos e análises.

## Stack tecnológica
- Frontend: React com Next.js
- Estilização: Tailwind CSS
- Banco de dados: Supabase (PostgreSQL)
- Autenticação: Supabase Auth
- Gráficos: Recharts
- Deploy: Vercel
- Ícones: Lucide React

## Funcionalidades principais
1. Cadastro e login de residentes
2. Registro de cirurgias com procedimentos detalhados
3. Dashboard com gráficos analíticos
4. Seção dedicada para bloqueios de nervo periférico
5. Análise de falhas e evolução
6. Comparação entre residentes (perfis públicos)
